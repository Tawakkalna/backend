## How To Use

From your command line:

#### # Clone this repository

```bash
git clone https://github.com/Tawakkalna/backend.git
```

#### # Go into the frontend directory

```bash
cd backend
```

#### # Install the dependencies

```bash
composer install
```
    
```bash
npm install
```

#### # Run the app

```bash
php artisan serve
```
